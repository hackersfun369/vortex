API Reference

Base URL: https://vortex-api.guessmesir-007.workers.dev

Overview

Vortex is a free tunnel service. Expose any local port to the internet via a named subdomain. All responses are JSON. Authentication uses API keys prefixed with vrtx_.

Installation

curl -fsSL https://hackersfun369.github.io/vortex/install.sh | sh

Installs cloudflared and the vortex CLI. Supports Linux, macOS, Windows.

Authentication

Authenticated endpoints require an API key obtained at signup or via /my/recover-apikey.

Authorization: Bearer vrtx_yourkey

Owner-only endpoints require the OWNER_TOKEN secret set during Worker setup.

Authorization: Bearer YOUR_OWNER_TOKEN

Public Endpoints

GET/health

API health check and version.


      

      
{ "status": "ok", "version": "2.0.0", "ts": 1234567890 }

    


    

      
POST/tunnel/create

      
Create a new tunnel. Auth optional — required only if reserve: true.

      
      

        
        
        
        
        

      

      
{
  "subdomain": "myapp",   // optional — auto-generates if omitted
  "port": 3000,           // optional — informational
  "privacy": false,       // optional
  "reserve": true,        // optional — requires Authorization header
  "meta": {}              // optional
}

      
Response 201: returns token, token_hint, cloudflared_cmd, url.

    


    

      
GET/tunnel/:subdomain

      
Get public info about a tunnel. Returns 404 for private tunnels.

List all active public tunnels.

Authenticated Endpoints

    
Require Authorization: Bearer vrtx_yourkey. Your API key is shown at signup and recoverable via /my/recover-apikey.


    

      
POST/tunnel/heartbeat/:subdomain tunnel token

      
Update last_seen. Max 1/hour per subdomain. Called automatically by the CLI.

    


    

      
GET/tunnel/recover/:subdomain API key

      
Recover the full token for a tunnel you own.

      
      

        
        
        
        

      

    


    

      
GET/my/tunnels API key

      
Get your profile and all owned tunnels with full details.

      
      

        
        
        

      

    


    

      
GET/my/recover-apikey API key

      
Recover your full API key. Returns the decrypted key.

      
      

        
        
        

      

    


    

      
POST/my/apply-api-access API key

      
Apply for direct API access (reserved subdomains, programmatic use).

Error responses

    

      
StatusMeaning

      
400Bad request — invalid input or missing required field

      
401Authentication required or invalid token

      
403Forbidden — wrong token, suspended account, insufficient access

      
404Not found — tunnel doesn't exist, expired, or private

      
409Conflict — subdomain already taken

      
429Rate limited — includes retry_after seconds

      
500Internal server error

    


    
Tunnel status values

    

      
StatusMeaning

      
activeReceiving heartbeats, last seen within 7 days

      
inactiveNo heartbeat for 7–30 days, not yet expired

      
graceExpired but in 30-day grace period (reserved subdomains only)

      
expiredPast inactivity limit — subdomain freed
